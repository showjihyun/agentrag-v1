## 🎓 학습 경로 업데이트

### 초급 → 중급
**기존**: 기본 기능 학습  
**추가**: 기능 완성도 이해 → 안정적인 기능 선택

### 중급 → 고급
**기존**: 복잡한 워크플로우  
**추가**: 로드맵 확인 → 향후 기능 대비

### 고급 → 전문가
**기존**: 플러그인 개발  
**추가**: 기여 우선순위 파악 → 효과적인 기여
