이미지를 매우 정확하게 분석하여 다음을 수행해주세요:

1. **표(테이블) 데이터 추출** (표가 있는 경우) - 최우선:
   - 모든 행과 열의 내용을 빠짐없이 추출
   - 병합된 셀도 정확히 표현
   - 헤더와 데이터를 명확히 구분
   - 복잡한 중첩 표도 구조 유지
   - 마크다운 표 형식으로 변환
   
   특히 장비 사양서의 경우:
   - 항목명과 사양값을 정확히 매칭
   - CPU, GPU, RAM, 저장장치 등 모든 항목 추출
   - 모델명, 용량, 규격 등 세부 정보 포함

2. **차트/그래프 분석** (차트가 있는 경우):
   - 차트 유형 식별
   - 모든 데이터 포인트와 값 추출
   - 추세와 패턴 설명

3. **일반 텍스트 추출**:
   - 제목, 부제목, 설명, 주석 등 모든 텍스트
   - 문서 번호, 날짜, 버전 정보

4. **이미지 설명**:
   - 문서의 목적과 주요 내용 요약

출력 형식:
---
[표 데이터] (표가 있는 경우)
| 항목 | 사양 | 비고 |
|------|------|------|
| CPU | Intel Core i7 | 8코어 |
| RAM | 32GB DDR4 | - |

또는 복잡한 표의 경우:
항목: CPU
- 모델: Intel Core i7-12700K
- 코어: 8코어 16스레드
- 클럭: 3.6GHz

항목: RAM
- 용량: 32GB
- 타입: DDR4
- 속도: 3200MHz

[차트 분석] (차트가 있는 경우)
- 차트 유형: (막대/선/원형 등)
- 주요 데이터: ...

[일반 텍스트]
(제목, 문서번호, 날짜 등)

[이미지 설명]
(전체 요약)
---

중요: 표의 모든 셀 내용을 빠짐없이 추출하고, 특히 장비 사양의 경우 모든 항목과 값을 정확히 매칭하여 추출해주세요.
