# Setup Scripts

Scripts for initial project setup and environment configuration.

## Windows Setup
- `setup.bat` - Main Windows setup script
- `setup-windows.bat` - Windows-specific setup
- `full-setup.bat` - Complete setup with all dependencies
- `complete-installation.bat` - Final installation steps

## Linux/Mac Setup
- `setup.sh` - Main Unix setup script
- `start-linux.sh` - Linux startup script
- `stop-linux.sh` - Linux shutdown script

## Database Setup
- `setup-postgres.bat` / `setup-postgresql.bat` - PostgreSQL setup
- `init-database.bat` - Database initialization
- `create-database.sql` - Database creation SQL

## Service Setup
- `setup-ollama.bat` - Ollama LLM setup
