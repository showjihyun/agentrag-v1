# Maintenance Scripts

Scripts for system maintenance and troubleshooting.

## Restart Scripts
- `restart-backend.bat` - Restart backend
- `restart-backend-simple.bat` - Simple restart
- `restart-backend-fixed.bat` - Restart with fixes
- `restart-ollama.bat` - Restart Ollama service
- `force-restart-backend.bat` - Force restart

## Fix Scripts
- `fix-dependencies.bat` - Fix Python dependencies
- `fix-env-encoding.bat` - Fix environment encoding
- `fix-warnings.bat` - Fix common warnings
- `fix_milvus.bat` / `fix_milvus.sh` - Fix Milvus issues

## Docker
- `docker-services-only.bat` - Start only Docker services
- `recreate-docker-containers.bat` - Recreate containers
- `reset-and-start.bat` - Reset and restart

## Database
- `merge-migrations.bat` - Merge Alembic migrations
