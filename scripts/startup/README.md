# Startup Scripts

Scripts for starting application services.

## All Services
- `start-all.bat` - Start all services
- `quick-start.bat` - Quick start for development
- `start-services.bat` - Start Docker services

## Backend
- `start-backend.bat` - Start backend server
- `start-backend-simple.bat` - Simple backend start
- `start-backend-fixed.bat` - Backend with fixes applied
- `start_backend.bat` - Alternative backend start

## Frontend
- `restart-frontend.bat` - Restart frontend

## Production
- `start_production.sh` - Production startup
- `start-windows.bat` - Windows production start
- `start-windows-debug.bat` - Windows debug mode
