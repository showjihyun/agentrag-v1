# Installation Scripts

Scripts for installing dependencies.

## Python Dependencies
- `install-dependencies-clean.bat` - Clean install
- `install-missing-packages.bat` - Install missing packages
- `install-grpcio-first.bat` - Install gRPC first (for Milvus)
- `install-pymilvus.bat` - Install PyMilvus

## Frontend Dependencies
- `install-frontend-deps.bat` - Install frontend packages

## ML/AI Dependencies
- `install-paddle-gpu.bat` - Install PaddleOCR with GPU
- `install-paddleocr-latest.bat` - Latest PaddleOCR
- `install-tool-dependencies.bat` - Tool dependencies
